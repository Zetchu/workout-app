# Reps & Routines (Workout Tracker)

A mobile workout companion application built with React Native and Expo. The app allows users to explore a collection of daily workouts and target specific muscle groups to optimize their training routines.

## Potential Features

- **Exercise Discovery:** Scroll and filter through an extensive catalog of movements and targeted exercises.
- **Dynamic Weekly Planner:** Build and save custom routines mapped out across the days of the week.
- **Smart Personalization:** Generate tailored workout recommendations based on user-specific body metrics, height, weight, and fitness goals.
- **History Tracking:** Log sets, reps, and weight progression over time to keep track of personal records.
